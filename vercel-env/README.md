Import each file into its matching Vercel project using the "Import .env" button.

- web.env -> apps/web project
- auth-service.env -> apps/auth-service project
- resource-service.env -> apps/resource-service project
- intelligence-service.env -> apps/intelligence-service project
- recommendation-service.env -> apps/recommendation-service project
- routing-service.env -> apps/routing-service project (optional/legacy)

Apply to both Production and Preview environments.
